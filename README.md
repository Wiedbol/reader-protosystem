# library-protosystem**(未完成)**
一个图书馆项目

## 功能

1.  导入图书
	1. 支持的格式：txt，epub，pdf
2. 阅读图书
	1. 书签
	2. 个性化
		1. 字体
		2. 背景
	3. 批注，笔记，高亮
		1. 批量管理
		2. 导出笔记和高亮
	4. 目录
	5. 进度条
3. 收藏图书
4. 删除图书
5. 主界面展示图书信息
	1. 按最近阅读和导入顺序排序
	2. 按评分排序
	3. 更改布局方式（列表、平铺等）
6. 注册与登录

## 安装项目依赖
```
npm install
```
## 启动项目
```
npm start
```
